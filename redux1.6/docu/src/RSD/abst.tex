\begin{abstract}
  The ReDuX system is a programmer's workbench for programming
  term rewriting systems.
  The documentation is meant as a guide for programmers who want to use the 
  collection of algorithms contained in the ReDuX system.
  It provides descriptions of all data structures, describes
  all important algorithms and module interfaces and 
  presents the programming conventions used in ReDuX programs.
\end{abstract}
